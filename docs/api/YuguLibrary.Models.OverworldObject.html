    runDelay

    Amount of delay (in seconds) that the unit undergoes during movement.

    Declaration
    public float runDelay
    Field Value
    Type Description
    Single
    Remarks

    Delay is enacted when moving across tiles in the X and Y axes. Lower moveSpeed values makes the unit move faster. For player units, the delay activates after moving to a new tile. For non-player units, the delay activates before trying to move; if the tile becomes occupied after the delay ends, the unit does not move.
