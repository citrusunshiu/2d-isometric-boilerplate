    Class HookFunction

    Grants Unit objects dynamic functionality at certain points during encounters.

    Assembly: Assembly-CSharp.dll

    Remarks

    Defines a specific DelegateFlags value and a snippet of code to attach to a Unit object. The code snippet is ran whenever the associated delegate flag function is ran by EncounterController.
